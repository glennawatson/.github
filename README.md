# .github

Default community health files for repositories under [@glennawatson](https://github.com/glennawatson).

GitHub falls back to the files in this repository (e.g. `FUNDING.yml`, issue templates, `CODE_OF_CONDUCT.md`) for any repo that doesn't define its own. See [GitHub's docs on default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

My GitHub profile README lives in [glennawatson/glennawatson](https://github.com/glennawatson/glennawatson).
